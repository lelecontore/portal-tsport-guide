# portal-tsport-guide

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖信息展示、导航辅助与功能指引等类别。所有页面均为静态文件，不依赖后端服务，便于直接访问与分发。

## 项目简介

portal-tsport-guide 是一个以 HTML 页面为主的归档仓库，旨在集中存放各类独立页面资源。仓库内容不针对特定域名或具体网站，而是提供通用性较强的页面模板与功能模块，供有需要的用户参考或使用。

页面内容以静态 HTML 编写，结构清晰，兼容主流浏览器，无需复杂配置即可在本地或托管环境中打开。

## 目录说明

```
portal-tsport-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── index.html      # 仓库首页入口
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，页面间无强耦合关系。
- `assets/`：存放页面通用的样式、脚本与媒体资源，便于统一管理与更新。

## 页面归档说明

仓库中的 HTML 页面按照功能或用途进行分类归档，每个页面均附带简要说明（位于页面头部注释或同级说明文件中）。页面可能包含以下类型：

- 信息展示页
- 导航辅助页
- 功能工具页
- 其他静态内容页

所有页面均为独立文件，可单独使用，不依赖其他页面或外部服务（CDN 资源除外，如有引用会在页面内注明来源）。

## 维护说明

- 新增页面请放置在 `pages/` 目录下，并保持目录结构整洁。
- 公共资源请统一存放至 `assets/`，避免在各页面中重复引入。
- 页面更新时请同步更新页面内版本注释或修改记录。
- 如需删除或废弃页面，建议将其移至 `archived/` 目录或添加明确废弃标记。

欢迎提交 Issue 或 Pull Request 进行内容补充与优化。所有贡献请遵循本仓库的目录结构与代码规范。
